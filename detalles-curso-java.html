Java desde Cero

Java es el lenguaje de programación para el backend con
mayor demanda laboral en grandes empresas de todo el mundo.

¿Que aprenderás?

  • Conocer la sintaxis y el flujo de trabajo de Java.
  • Aprenderemos los tipos de datos básicos, condicionales, ciclos del
    lenguaje y también aprenderemos a compilar el código fuente.


Requisitos previos: conocimientos básicos de los conceptos de
programación.


Tiempo de dedicación necesario: 12 horas
